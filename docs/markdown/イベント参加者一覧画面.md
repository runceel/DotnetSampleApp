イベント参加者一覧画面は、イベントの受付で参加者一覧の出欠の確認を行うための画面です。

イベント参加者は以下の項目を持ちます。

## 1. データ定義

イベント参加者 (Attendee)

| 項目名 | 型名 | 初期値 | 説明 |
|--------|------|--------|------|
| ID | int | 0 | イベント参加者の一意識別子。0 から 1 ずつインクリメントされる。 |
| AccountName | string | "" | 参加者のアカウント名 |
| IsAttended | bool | false | 出欠したかどうか。true が出席、false が欠席や、まだ受付前の状態 |

## 2. 画面項目定義

| 項目名 | コントロール名 | 初期値 | イベント |
|--------|----------------|--------|----------|
| アカウント名検索 | TextBox | 空文字 | タイピング時にフィルタリング |
| 参加者一覧 | QuickGrid | 参加者一覧 | |
| 　AccountName ヘッダー | Text | アカウント名 | |
| 　IsAttended ヘッダー | Text | 参加・不参加 | |
| 　AccountName | Text | AccountName の値 | |
| 　IsAttended | CheckBox | IsAttended の値 | クリック時にデータを保存 |
